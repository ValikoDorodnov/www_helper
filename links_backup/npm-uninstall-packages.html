Удаление пакетов npm с помощью `npm uninstall`

Как удалить пакет npm Node локально или глобально

Чтобы удалить ранее установленный пакетлокально(с помощьюnpm install <package-name>вnode_modulesпапка, запустить

npm uninstall <package-name>

из корневой папки проекта (папки, содержащей папку node_modules).

С использованием-Sфлаг, или--save, эта операция также удалит ссылку вpackage.jsonфайл.

Если пакет был зависимостью разработки, перечисленной в devDependencies файлаpackage.jsonфайл, вы должны использовать-D/--save-devфлаг, чтобы удалить его из файла:

npm uninstall -S <package-name>
npm uninstall -D <package-name>

Если пакет установленглобально, вам нужно добавить-g/--globalфлаг:

npm uninstall -g <package-name>

Например:

npm uninstall -g webpack

и вы можете запустить эту команду из любого места в вашей системе, потому что папка, в которой вы сейчас находитесь, не имеет значения.
